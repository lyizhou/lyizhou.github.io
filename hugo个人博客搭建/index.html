Hugo个人博客搭建

Publish date: Jul 10, 2022
Tags:
  • 博客搭建

创建Github库

  1. 打开Github，新建库
  2. 项目名与Github用户名保持一致，例如我的用户名为lyizhou，项目名为lyizhou.github.io，勾选Readme。

安装Hugo和Git

  1. 下载Windows版本的Hugo进行安装
  2. 解压后，添加环境变量，例如 D:\hugo_0.101.0_Windows-64bit
  3. 打开命令行， 输入hugo version，检查是否安装成功

新建Hugo网站

  1. 新建目录用于存放Hugo网站的文件
  2. 打开命令行，切换到该目录，执行命令新建一个Hugo网站
hugo new site blog

Hugo主题

  1. 打开hugo theme的网站，选择主题，在这里选择hugo-theme-even
  2. 下载主题文件到本地 
    # 进入网站blog目录
cd blog
# 进入themes目录
cd themes
# 下载主题文件到hugo-theme-even目录下
git clone https://github.com/olOwOlo/hugo-theme-even.git hugo-theme-even
  3. 在Hugo根目录下配置config.toml
  4. 在blog目录下备份原来的config.toml文件，然后将D:\hugo_0.101.0_Windows-64bit\myblog\themes\even复制病覆盖原来的文件，修改配置。

新建文章

  1. 进入blog目录下，选择 Git Bash Here
  2. 新建文章 
    hugo new post/test.md
  3. 编辑文章内容并保存
  4. 启动Hugo server 
    hugo server -D
  5. 打开 http://localhost:1313/可以进行预览

部署

  1. 构建Hugo网站 在Hugo网站blog目录下执行hugo命令构建 Hugo会将构建的的网站内容默认保存在根目录下的public/文件下

  2. 将网站与Github建立联系

    • 进入public目录下，初始化Git库
    git init
    • 将git本地库关联至远程库
    git remote add origin git@github.com:lyizhou/lyizhou.github.io.git

  3. 提交修改 在public目录下，通过一下命令提交

    # 查看当前修改状态
git status
# 添加修改过的文件， . 表示所有，也可以指定文件
git add .
# ""里面的信息就是提交的说明内容
git commit -m "commit1"

  4. 提交修改至远程仓库 在public目录下，通过如下命令提交

    git push -u origin master

  5. 在Github页面将Source branch 改为master

文章操作

  1. 新建文章 
    # 1. 新建一篇文章，在blog目录
hugo new post/test2.md
# 2. 在blog\content\post目录下找到对应的文件进行修改
# 3. 本地预览 http://localhost:1313/，可以一边预览一边修改
hugo server -D
# 4. 构建Hugo网站
hugo
# 5. 切换至public目录，提交本地修改至本地库
git add .
git commit -m "commit init"
# 6. 提交修改至远程库
git push -u origin master
  2. 修改文章 
    # 1. 在blog\content\post找到对应文件
进行修改
# 2. 本地预览 http://localhost:1313/
hugo server -D
# 3. 构建hugo网站
hugo
# 4. 切换到public目录，提交修改至本地库
git add .
git commit -m "modification"
# 5. 将修改推至远程库
git push -u origin master
  3. 删除文章 
    # 1. 在blog\content\post以及blog\public两处找到对应文件
# 2. 删除两处文件
# 3. 本地预览 http://localhost:1313/
hugo server -D
# 4. 构建hugo网站
hugo
# 5. 切换到public目录，提交修改至本地库
git add .
git commit -m "modification"
# 6. 将修改推至远程库
git push -u origin master
  4. 添加图片 将图片放在static文件下，以static为相对路径，然后进行引用，引用方式为：![](/imgs/doge.jpg)
    1. 在文章目录下新建文件夹存放照片，引用时采用相对路径。
    2. 完成后在引用路径前加/
    3. 之后将图片文件夹移动到static目录下即可。
